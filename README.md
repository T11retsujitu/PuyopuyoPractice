# ぷよぷよ練習 (PuyopuyoPractice)

一手ごとに**完全に停止する**ぷよぷよ。時間に追われずじっくり考えて、置いた手の
**メリット・デメリットの解説**と**他の有力候補**から「なぜその一手なのか」を学べる練習ツールです。

> このツールは「正解を当てるツール」ではなく「考え方を学ぶツール」です。
> ぷよぷよの一手に正解はひとつとは限りません。評価は常に「候補との比較」として提示され、
> 正解/不正解を断定しません。

## 機能

- **とこぷよ練習モード** — 対戦・おじゃま・時間制限なし。一手置くごとに完全停止し、
  自分のタイミングで次のツモへ進めます。「待った」(undo) もいつでも可能。
- **一手ごとの評価** — 全設置候補をシミュレーションし、あなたの一手を
  S/A/B/C(最有力候補とのスコア差)で評価。メリット○・デメリット△を日本語で解説し、
  他の有力候補も縮小盤面つきで提示します。
- **練度別の評価・解説** — 初心者(平積み・連結・ちぎり回避)/中級者(連鎖ポテンシャル・発火点)/
  上級者(発火点の柔軟性・効率)で評価基準と解説のトーンが変わります。
- **土台練習モード** — 階段積み・カギ積み・GTR を**実戦同様のランダムツモ**で練習。
  左右どちらの型かは盤面から自動判定し、完成度バー・お手本オーバーレイ・
  「形に合致/崩し」のフィードバックを表示。土台に合わないツモは「逃がし」を正しく評価します
  (ツモ運では罰しません)。
- **設置前ヒント**・**連鎖アニメーション(速度変更/オフ)**・**シード指定**
  (同じシード = 同じツモ順で練習のやり直しが可能)。

## 操作

| キー | 動作 |
|---|---|
| ← → | 移動 |
| ↑ / X | 右回転 |
| Z | 左回転 |
| ↓ / Space / Enter | 設置 |
| N / Enter | 次のツモへ(停止中) |
| U | 待った(1手戻す) |
| H | ヒント表示切替 |

マウス/タッチ: 列をポイントして移動、クリックで設置、ホイールで回転。画面のボタンでも操作できます。

## ルール(ぷよぷよ通準拠)

6列×13段(13段目は幽霊段)・4色・4個連結で消去・通の連鎖得点計算・
3列目12段目が埋まると窒息。ツモは128手ループ(4色均等、初手2手は3色以内)。
まわし等の高度な入力は対象外の簡易操作モデルです。

## 開発

```bash
npm install
npm run dev        # 開発サーバ
npm test           # ユニットテスト(コア・評価エンジン・テンプレート・reducer)
npm run build      # 型チェック + プロダクションビルド
```

構成: `src/core`(純粋なゲームロジック)→ `src/eval`(ヒューリスティック評価・
土台テンプレート・解説文生成)→ `src/state` / `src/ui`(React)。
コアと評価エンジンは UI 非依存で、すべてユニットテストがあります。
